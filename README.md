# ライブラリの導入
`pip install tweepy`<br/>
`pip install requests_oauthlib`<br/>

# Botの動かし方
* Discordで共有したsecret.pyというファイルをtwitterフォルダ内に配置<br/>
* ターミナルで`py twitter/twitter.py`を実行<br/>
